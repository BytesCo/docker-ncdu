# Bytes.co Alpine ncdu

A no-frills Alpine ncdu container. Currently based on Alpine 3.17.2. Small and secure. Helpful when you run out of disk space.

## Example Usage
```bash
docker run --rm -it -v /:/mnt/root bytesco/ncdu /mnt/root
```
